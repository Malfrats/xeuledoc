# xeuledoc
![PyPI](https://img.shields.io/pypi/v/xeuledoc) ![PyPI - License](https://img.shields.io/pypi/l/xeuledoc)

Fetch information about any public Google document.\
It's working on :
- Google Docs
- Google Spreadsheets
- Google Slides
- Google Drawning
- Google My Maps
- Google Apps Script
- Google Jamboard
